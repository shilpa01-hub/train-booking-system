# train-booking-system
# 🚆 Railway Ticket Booking System

A full-stack railway reservation system inspired by IRCTC, developed using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript. The application allows users to search trains, book tickets, manage bookings, and make secure online payments through an intuitive interface.

##  Features

-  User Authentication & Authorization
-  Train Search & Booking
-  Date-wise Seat Availability
-  Automatic Seat Allocation
-  Waiting List Management
-  Secure Payment Integration
-  Admin Dashboard for Train & Booking Management
-  Responsive User Interface
-  Booking History & Ticket Details
